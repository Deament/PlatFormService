# 项目介绍

- 该项目主要结合dubbox和springboot框架进行搭建
- 通过maven构建项目，实现多profile配置
- 前后端完全分离，该项目只实现后端设计，前端内容后续添加
- 采用spring security对角色权限进行管理
- 使用jwt协议实现客户端调用的安全策略
- 利用mybatis实现数据的读写分离
- 通过redis进行缓存
- 使用logback记录日志
- 实现项目水平扩展及高可用的架构
